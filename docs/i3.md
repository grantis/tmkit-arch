# i3 Window Manager

i3 is a tiling window manager. Instead of dragging windows around,
they automatically tile to fill your screen. You control everything
with the keyboard. This guide is for bare Linux installs with a display
(not WSL).

---

## When to use this

- You have a full Linux install (Arch, Ubuntu, Debian) with a monitor
- You want a minimal, fast desktop that uses almost no resources
- You're comfortable with the terminal and want to go deeper

Skip this if you're on WSL - you don't need it there.

---

## Install (with all the pieces from the video)

```bash
# Ubuntu/Debian
sudo apt install -y i3 i3status i3lock xorg alacritty rofi picom feh

# Arch
sudo pacman -S i3-wm i3status i3lock xorg-server alacritty rofi picom feh
```

| Package | What it does |
|---|---|
| `i3` | The window manager |
| `xorg` | Display server (draws things on screen) |
| `alacritty` | Fast GPU-accelerated terminal |
| `rofi` | App launcher (like Spotlight on Mac) |
| `picom` | Compositor (shadows, blur, transparency) |
| `feh` | Sets wallpaper |
| `i3status` | Status bar content (time, battery, etc) |

---

## Start i3

If you installed from a display manager (GDM, LightDM) just select i3 at login.

For a bare server install, add to `~/.xinitrc`:

```bash
exec i3
```

Then start with: `startx`

## Essential Keybindings

The default modifier key (`$mod`) is the Windows/Super key.

### Basics

| Key | Action |
|---|---|
| `$mod+Enter` | Open terminal |
| `$mod+d` | Open app launcher (rofi) |
| `$mod+Shift+q` | Close window |
| `$mod+Shift+e` | Exit i3 |
| `$mod+Shift+r` | Reload i3 config |

### Window navigation

| Key | Action |
|---|---|
| `$mod+h/j/k/l` | Focus left/down/up/right |
| `$mod+Shift+h/j/k/l` | Move window |
| `$mod+f` | Fullscreen toggle |
| `$mod+e` | Split layout (default) |
| `$mod+s` | Stacked layout |
| `$mod+w` | Tabbed layout |

### Workspaces (virtual desktops)

| Key | Action |
|---|---|
| `$mod+1` to `$mod+9` | Switch to workspace |
| `$mod+Shift+1` to `9` | Move window to workspace |

## Minimal i3 Config

Location: `~/.config/i3/config`

```
# Modifier key (Super/Windows key)
set $mod Mod4

# Font
font pango:JetBrainsMono Nerd Font 10

# Terminal
bindsym $mod+Return exec alacritty

# App launcher
bindsym $mod+d exec rofi -show drun

# Kill window
bindsym $mod+Shift+q kill

# Reload config
bindsym $mod+Shift+r restart

# Exit i3
bindsym $mod+Shift+e exec i3-nagbar -t warning -m 'Exit i3?' -B 'Yes' 'i3-msg exit'

# Window navigation
bindsym $mod+h focus left
bindsym $mod+j focus down
bindsym $mod+k focus up
bindsym $mod+l focus right

# Move windows
bindsym $mod+Shift+h move left
bindsym $mod+Shift+j move down
bindsym $mod+Shift+k move up
bindsym $mod+Shift+l move right

# Workspaces
bindsym $mod+1 workspace 1
bindsym $mod+2 workspace 2
bindsym $mod+3 workspace 3
bindsym $mod+4 workspace 4
bindsym $mod+5 workspace 5

bindsym $mod+Shift+1 move container to workspace 1
bindsym $mod+Shift+2 move container to workspace 2
bindsym $mod+Shift+3 move container to workspace 3

# Gaps (if using i3-gaps)
gaps inner 8
gaps outer 4

# Wallpaper
exec_always feh --bg-scale ~/wallpaper.jpg

# Compositor
exec_always picom -f

# Status bar
bar {
  status_command i3status
  position top
  font pango:JetBrainsMono Nerd Font 10
}
```

## Minimal Alacritty Config

Location: `~/.config/alacritty/alacritty.toml`

```toml
[font]
normal = { family = "JetBrainsMono Nerd Font", style = "Regular" }
size = 13.0

[window]
padding = { x = 12, y = 12 }
opacity = 0.95
decorations = "none"

[colors.primary]
background = "#1a1b26"
foreground = "#c0caf5"
```

## Workspace Layout Tips

The video suggests dedicating workspaces:

- `1` - Terminal / main work
- `2` - Browser (Firefox)
- `3` - Notes (neovim)
- `4` - Tasks (taskwarrior-tui)
- `5` - Music / misc

Switch instantly with `$mod+number`. No mouse needed.
